| Name | Size | Line Height | Usage |
|------|------|-------------|-------|

## 🎯 Logo Usage

### Clear Space
- Minimum clear space: 16px on all sides
- Never distort or rotate the logo
- Maintain aspect ratio when scaling

### Color Variants
- **Full Color**: Use on light backgrounds
- **White**: Use on dark/colored backgrounds
- **Monochrome**: Use when color isn't available

## 🌓 Dark Mode

All colors have dark mode variants:

- Backgrounds become dark charcoal (#0A0A0A)
- Text inverts to light colors
- Primary/accent colors remain vibrant
- Reduce contrast to prevent eye strain

## ♿ Accessibility

- Minimum contrast ratio: **4.5:1** for normal text
- Minimum contrast ratio: **3:1** for large text
- Focus states must be visible
- Never rely on color alone to convey meaning
